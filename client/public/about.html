About the Movie Database

This application aims to provide a database of movies along with relevant information such as the rating, budget, review score, actors, etc. Users can browse movies, favorite them, and even add their own. This app will be a useful tool in discovering new movies as well as keeping a history of previously watched ones. The database is created by our group members Kavi, Jacky, Henry, and Josue.

We used Heroku, HTML, CSS, AND Javascript to create our database.
